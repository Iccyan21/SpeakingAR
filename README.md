# 概要
英語で相手が喋ってきた場合、それを翻訳するのとその喋ったことに対し、AIが自動でどう返答するべきかと返答する文章の発音をかたがなで表記する

OPENAIのAPIKEYだけ必須

# 使い方
録音ボタンを押して相手の発音を録音する

音声取得後、自動で表示される (録音Buttonは押したままで)

# 参考画像
Hello nice to meet you 
相手が喋った文章

意味:
相手が喋った意味

返し方:
相手が喋った文章に対して、何を返答すべきかの提案

読み方
返答すべきかの提案英文の発音をかたがなで読み方を表示する
